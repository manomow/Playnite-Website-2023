Playnite is an open source video game library manager with one simple goal

To provide a unified interface for all of your games.

Download

Windows 7 and newer supported • Changelog

Import popular libraries

Link games from Steam, Epic, EA, GOG, Battle.net, Ubisoft Connect & other services, including games that are not installed!

Emulation support

Playnite supports a wide variety of console emulators.

Playtime counter

Whether it's a modern title or an emulated classic, Playnite allows you to easily keep track of how much time you've spent in-game.

Fullscreen mode

Experience your games in the best possible way with Playnite's fullscreen mode, complete with full controller support.

Extend & Customize Playnite with Add-ons

Add-ons can integrate with your game libraries and metadata sources, while also providing a range of additional features.

Whether you're looking to change Playnite's appearance, modify its layout, or adjust its functionality, Add-ons tailor Playnite to your unique preferences.

Fast

Playnite's UI is fast and light on resources, even with large libraries.

Completely Free

No Playnite features are locked behind a paywall & the complete source code is available under the MIT license.

Portable

Playnite can run from portable storage without need of installing system dependencies.

IGDB Integration

Playnite can automatically retrieve metadata for your games, including covers, descriptions, and other information from sources like IGDB.com. Add-ons can extend this functionality to other sources as well.

Frequently Asked Questions

What does Playnite’s name mean?

Playnite is a combination of Play and Unite

Will there be Linux or macOS version?

Not in near future, mainly for technical reasons. More information here.

Does Playnite support X controller?

Gamepad-Controller-support

Can I link multiple accounts?

You can't, unless a library integration plugin explicitly supports it (gives you an option to link multiple accounts)

You can however link one account, import games, link a different account, import games again and you'll end up with a library that contains games from both accounts. Playnite never automatically removes games from the library so linking a different account won't affect games from other accounts in any way.
